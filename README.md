# MyTipsy
술값, 밥값 등 회비를 랜덤을 몰아서 내거나, N분의 1 할 수 있는 앱

## Requirements
- Language
    - Swift
- Framework
    - UIKit
    
## MVP
- 몰아서 내기 (몰빵)
- 나눠서 내기 (N빵)

## Architecture
- MVC
